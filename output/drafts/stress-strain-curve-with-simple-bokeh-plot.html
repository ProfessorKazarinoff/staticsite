Stress Strain Curve in Python using a simple bokeh plot

Date Thu 21 December 2017 Tags python / matplotlib / engineering / jupyter / notebook / bokeh / altair

Code pulled from a jupyter notebook using liquid tags

In [3]:
from bokeh.plotting import figure, output_file, show
import pandas as pd
output_file("line.html")

p = figure(plot_width=400, plot_height=400)
df = pd.read_csv('stress_strain_data.csv', sep=',', header=None, skiprows=0)
df.columns = ['strain', 'stress']

# add a line renderer
p.line(df['strain'], df['stress'], line_width=2)

#show(p)
from IPython.core.display import HTML
HTML('line.html')
