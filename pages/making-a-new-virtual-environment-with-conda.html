Making a new virtual environment with Conda

To start a new project, it is best pratice to create a new virtual environment. I have the Anaconda distribution of python installed on machine at work. Using the conda prompt is a bit like using the terminal on a Mac or Linux. To start it go to the windows start button on the lower left and select anaconda prompt. To make the new environment we need to use the following commands:

C:\Users\user.name> conda create --name webscrape python=3.6

The following NEW packages will be INSTALLED:

    certifi:        2016.2.28-py36_0
    pip:            9.0.1-py36_1
    python:         3.6.2-0
    setuptools:     36.4.0-py36_0
    vs2015_runtime: 14.0.25420-0
    wheel:          0.29.0-py36_0
    wincertstore:   0.2-py36_0

Proceed ([y]/n)? y

#
# To activate this environment, use:
# > activate webscrape
#
# To deactivate an active environment, use:
# > deactivate
#
# * for power-users using bash, you must source
#
C:\Users\user.name> activate webscrape

(webscrape) C:\Users\peter.kazarinoff>
````

If you see the ```(webscrape)``` in parenthesis before the command prompt, that means you set up the new virtual environment and are now using it. You can get a list of your virtual environments using the ```conda env list``` command.

```terminal
pelican                  C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3\envs\pelican
webscrape             *  C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3\envs\webscrape
root                     C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3

Notice the * asterisk on the line with webscrape. The virtual environment with the * is the currenly active environment.

To exit the virtual enviroment, use the command deactivate. If you do this and then run conda env list again, you will see that there is no * because you are not currently usinging one of the virtual environments. We can spin up the virtual environment again with the activate command.

(webscrape) C:\Users\peter.kazarinoff>deactivate

C:\Users\peter.kazarinoff> conda env list

pelican                  C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3\envs\pelican
webscrape                C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3\envs\webscrape
root                     C:\Users\peter.kazarinoff\AppData\Local\Continuum\Anaconda3

C:\Users\peter.kazarinoff> activate webscrape
(webscrape) C:\Users\peter.kazarinoff>
